# 이론적 설명

## 주사위가 2개일때
> 내가 선택한 전략

1. dice1∈{4,5,6}이면 dice1을 십의 자리로 사용
2. dice1∈{1,2,3}이면 dice1을 일의 자리로 사용
+ dice2는 자동적으로 남은자리에 고정됌

### 첫번째 경우
> dice1∈{4,5,6}이면 dice1을 십의 자리로 사용

+ 식 : Y = 10⋅dice1+dice2
+ 두번째 주사위의 값은 1~6까지만 가능하므로 이를 이용해서 기댓값을 구해본다면?

+ E[Y∣dice1=a]= 10a + 3.5
