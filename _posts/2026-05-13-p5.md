# 1. 군집의 임상적 특성표 

## 1-1. 분석 목적

- 최종 baseline 체성분 군집이 실제로 어떤 임상적·생활습관적 특성을 가지는지 확인하기 위해 군집별 임상특성표를 작성하였다.
- 군집은 최종 baseline K-means 3군 결과를 사용하였다.

- 근육우세·저지방형: 8명
- 복부비만·고지방형: 13명
- 저근육형: 13명

총 분석대상자는 34명이다.

---

## 1-2. 표 작성 기준

### 연속형 변수
- 제시 형식: 중앙값 [Q1, Q3]
- 군집 간 비교: Kruskal–Wallis 검정

### 범주형 변수
- 제시 형식: n (%)
- 군집 간 비교: Chi-square 검정

### 결측 처리
- 걸음수와 수면은 주분석 기준과 동일하게 median imputation 자료를 사용하였다.
- 대체 순서:
  1. 동일 대상자-visit 중앙값
  2. 동일 대상자 중앙값
  3. 동일 visit 중앙값
  4. 전체 중앙값

---

## 1-3. 군집별 임상특성표 핵심 결과

### 1) 연령, 유병기간, 기저 혈당상태

군집 간 다음 변수들은 유의한 차이가 없었다.

| 변수 | p-value |
|---|---:|
| 나이 | 0.778 |
| 당뇨 유병기간 | 0.949 |
| Baseline HbA1c | 0.267 |

즉, 세 군집은 연령, 당뇨 유병기간, 초기 HbA1c 수준이 크게 다른 집단으로 보이지 않았다.

---

### 2) 체성분 관련 변수

다음 변수들은 모두 군집 간 유의한 차이를 보였다.

| 변수 | p-value |
|---|---:|
| BMI | <0.001 |
| 허리둘레 | <0.001 |
| 골격근량 | <0.001 |
| 체지방률 | <0.001 |
| 내장지방 | <0.001 |

이는 군집이 의도한 체성분 표현형을 잘 반영하고 있음을 보여준다.

#### 군집별 특징
- **근육우세·저지방형**
  - 골격근량이 가장 높음
  - 체지방률과 내장지방이 가장 낮음

- **복부비만·고지방형**
  - BMI, 허리둘레, 내장지방이 가장 높음
  - 체지방률도 높은 편

- **저근육형**
  - 골격근량이 가장 낮음
  - BMI와 허리둘레는 낮은 편이지만 체지방률은 높음

따라서 세 군집은 각각  
**근육량 중심의 건강한 체성분형**,  
**복부비만이 두드러지는 고지방형**,  
**근육량이 부족한 저근육형**으로 해석할 수 있다.

---

### 3) 성별 분포

성별 분포는 군집 간 유의하게 달랐다.

| 군집 | 남성 비율 |
|---|---:|
| 근육우세·저지방형 | 87.5% |
| 복부비만·고지방형 | 92.3% |
| 저근육형 | 15.4% |

- p < 0.001

즉, 저근육형은 여성 비율이 매우 높았고,  
근육우세·저지방형과 복부비만·고지방형은 남성 비율이 높았다.

이는 이후 군집별 혈당지표 비교 분석에서  
**성별 보정 여부가 중요한 쟁점이 되는 이유**를 뒷받침한다.

---

### 4) 약물, 걸음수, 수면

다음 변수들은 군집 간 유의한 차이가 없었다.

| 변수 | p-value |
|---|---:|
| Baseline medication type | 0.734 |
| 하루 평균 걸음수 | 0.424 |
| 하루 평균 수면시간 | 0.286 |

즉, 세 군집의 차이는 약물 복용 유형이나 기본적인 생활습관 차이로 단순하게 설명되지는 않았다.

---

### 5) 식사 섭취량

정식식사 1회당 에너지 및 탄수화물 섭취량은  
통계적으로 유의한 차이에 도달하지는 않았으나, 군집별 차이 경향이 관찰되었다.

| 변수 | p-value |
|---|---:|
| 정식식사 1회당 에너지 섭취량 | 0.053 |
| 정식식사 1회당 탄수화물 섭취량 | 0.068 |

#### 군집별 중앙값
- **에너지 섭취량**
  - 근육우세·저지방형: 675.5 kcal
  - 복부비만·고지방형: 627.1 kcal
  - 저근육형: 463.6 kcal

- **탄수화물 섭취량**
  - 근육우세·저지방형: 81.0 g
  - 복부비만·고지방형: 90.6 g
  - 저근육형: 60.6 g

저근육형은 식사 1회당 에너지와 탄수화물 섭취량이 가장 낮은 방향을 보였고,  
복부비만·고지방형은 탄수화물 섭취량이 가장 높은 방향을 보였다.

다만 p-value가 각각 0.053, 0.068이므로  
“유의한 차이가 있었다”기보다는  
**“군집 간 섭취량 차이의 경향이 관찰되었다”**고 표현하는 것이 적절하다.

## 1-4. 결과 해석

최종 baseline 체성분 군집은  
체성분 관련 지표에서는 매우 뚜렷한 차이를 보였고,  
이는 군집의 임상적 해석 가능성을 지지한다.

반면 나이, 당뇨 유병기간, baseline HbA1c, 약물 유형, 걸음수, 수면시간은 군집 간 차이가 유의하지 않았다.  
따라서 이후 CGM 지표 분석에서 군집 차이가 확인된다면,  
그 차이는 단순히 연령이나 기저 HbA1c, 생활습관 수준의 차이만으로 설명되기는 어렵다.

다만 성별 분포는 군집 간 매우 크게 달랐으므로,  
이후 GEE daily-level 분석에서  
**성별을 포함한 모델과 제외한 모델을 함께 제시하는 전략은 타당하다.**

# 2. 군집 안정성 분석

- 체성분 군집의 재현성과 안정성을 평가하기 위해 군집 알고리즘 간 일치도 분석, Bootstrap clustering stability analysis, Leave-one-out sensitivity analysis를 수행하였다.

## 2-1. 군집 알고리즘 간 일치도

- 최종 군집해로 선택한 K-means k=3과 대안 군집방법인 Ward 계층적 군집분석 k=3의 일치도를 평가한 결과, ARI = 0.899, NMI = 0.902로 매우 높은 수준의 일치도를 보였다.

- 이는 군집 형성 방법이 달라져도 유사한 3군 구조가 재현되었음을 의미한다.

---

## 2-2. Bootstrap clustering stability analysis

- 표본 재추출에 따른 군집 구조의 안정성을 확인하기 위해 500회 bootstrap resampling을 수행하였다.  
- 각 반복에서 bootstrap 표본으로 K-means k=3을 다시 적합한 뒤, 원래 전체 표본에서 얻은 기준 군집해와의 일치도를 ARI로 평가하였다.

| 지표 | 값 |
|---|---:|
| 반복 횟수 | 500 |
| ARI 평균 | 0.634 |
| ARI 표준편차 | 0.245 |
| ARI 최솟값 | 0.091 |
| ARI 25% 분위수 | 0.436 |
| ARI 중앙값 | 0.652 |
| ARI 75% 분위수 | 0.849 |
| ARI 최댓값 | 1.000 |
| ARI 2.5% 분위수 | 0.223 |
| ARI 97.5% 분위수 | 1.000 |

- Bootstrap 분석에서 ARI의 중앙값은 0.652로 나타나, 재표본추출 상황에서도 군집 구조가 전반적으로 중등도 수준에서 유지되는 경향을 보였다.

- 다만 ARI 분포의 범위가 비교적 넓게 나타났는데, 이는 본 연구의 baseline 군집분석 표본 수가 34명으로 제한적이고, 일부 군집의 크기가 상대적으로 작기 때문에 재표본 구성에 따라 군집 경계가 일부 변동할 수 있음을 시사한다.

- 따라서 본 군집은 완전히 고정된 구조라기보다는, 소표본 기반 탐색적 군집분석으로서 일정 수준의 안정성을 보인 군집해로 해석하는 것이 적절하다.
---

## 2-3. Leave-one-out sensitivity analysis

- 특정 단일 대상자가 전체 군집구조에 과도하게 영향을 미치는지 확인하기 위해 Leave-one-out sensitivity analysis를 수행하였다.

- 전체 34명의 대상자 중 1명을 순차적으로 제외한 뒤, 나머지 대상자들만을 이용해 K-means k=3 군집분석을 다시 수행하였다.  
- 이후 각 반복에서 얻어진 군집해와 원래 기준 군집해의 일치도를 ARI와 NMI로 평가하였다.

| 지표 | 값 |
|---|---:|
| 반복 횟수 | 34 |
| ARI 평균 | 0.950 |
| ARI 표준편차 | 0.117 |
| ARI 최솟값 | 0.644 |
| ARI 25% 분위수 | 1.000 |
| ARI 중앙값 | 1.000 |
| ARI 75% 분위수 | 1.000 |
| ARI 최댓값 | 1.000 |
| NMI 평균 | 0.964 |
| NMI 표준편차 | 0.083 |
| NMI 최솟값 | 0.757 |
| NMI 25% 분위수 | 1.000 |
| NMI 중앙값 | 1.000 |
| NMI 75% 분위수 | 1.000 |
| NMI 최댓값 | 1.000 |

- 이는 대부분의 경우 단일 대상자 1명을 제외하더라도 군집 구조가 거의 동일하게 유지되었음을 의미한다.  
- 즉, 본 연구의 체성분 군집은 특정 한 명의 관측치에 의해 임의적으로 형성된 결과라고 보기 어렵다.

- 일부 대상자를 제외했을 때 ARI가 다소 낮아지는 경우가 관찰되었으나, 전체적으로 leave-one-out 결과가 매우 높은 안정성을 보였다는 점에서 최종 군집해는 단일 대상자 수준의 영향에 대해서는 충분히 견고한 구조를 가진 것으로 판단된다.

---

## 2-4. 종합 해석

- 본 연구의 체성분 군집은 다음 세 가지 근거에서 타당성과 안정성이 뒷받침된다.

1. **내부 군집평가 지표에서 K-means k=3이 우수한 성능을 보였다.**
   - Silhouette score = 0.381
   - Davies-Bouldin index = 0.881

2. **대안 알고리즘인 Ward 계층적 군집분석과 매우 높은 일치도를 보였다.**
   - ARI = 0.899
   - NMI = 0.902

3. **추가적인 안정성 분석에서도 군집구조의 재현성이 확인되었다.**
   - Bootstrap stability: ARI 중앙값 = 0.652
   - Leave-one-out sensitivity: ARI 중앙값 = 1.000, NMI 중앙값 = 1.000

이를 종합하면,  
본 연구에서 사용한 **K-means k=3 체성분 군집해는 알고리즘 선택이나 특정 단일 대상자에 과도하게 의존하지 않았으며**,  
표본 재추출에 따른 일부 변동성은 존재하지만 **탐색적 군집분석으로서 수용 가능한 수준의 안정성을 보였다**고 해석할 수 있다.

# 3. Baseline CGM 조정 분석

## 3-1. 분석 목적

- 기존 daily-level GEE 분석에서는 V1, V4, V5, V6의 반복측정 자료를 이용하여 체성분 군집에 따른 CGM 지표 차이를 평가하였다.

- 다만 이 경우,

> “추적시점의 CGM 차이가 실제 군집 특성 때문이 아니라,  
> 이미 baseline V1에서 혈당 상태가 달랐기 때문은 아닌가?”

라는 비판이 가능하다.

- 이를 보완하기 위해 이번에는 **V4–V6 추적시점 CGM 지표를 결과변수로 두고, 동일 지표의 baseline V1 평균값을 추가 보정한 GEE 분석**을 수행하였다.

- 또한 기존 분석에서 기준군을 **근육우세·저지방형**으로 두고  
  - 복부비만·고지방형 vs 근육우세·저지방형
  - 저근육형 vs 근육우세·저지방형  
  을 비교하였다면,

  이번 보완 분석에서는 추가로

> **저근육형 vs 복부비만·고지방형**

의 직접 비교도 수행하였다.

- 이는 두 취약 체성분 유형 중 **어느 군집이 상대적으로 더 불리한 CGM 프로파일을 보이는지**를 탐색적으로 평가하기 위함이다.

---

## 3-2. 분석 구조

### 분석 대상

- 결과변수 분석 시점: **V4, V5, V6**
- 분석 단위: **등록번호 + visit + day**
- 최종 분석 표본:
  - daily row: **1,053개**
  - 대상자: **30명**
  - subject-visit: **85개**

### Baseline 보정 방식

- 각 outcome에 대해 **동일 CGM 지표의 V1 평균값**을 보정변수로 사용하였다.

예시:

| 추적 outcome | 보정한 baseline 변수 |
|---|---|
| Daily mean glucose | V1 daily mean glucose 평균 |
| Daily CV | V1 daily CV 평균 |
| Daily TIR 70–180 | V1 daily TIR 70–180 평균 |
| Daily TAR 180–250 | V1 daily TAR 180–250 평균 |
| Daily TAR >250 | V1 daily TAR >250 평균 |

- baseline CGM 변수는 모든 outcome에서 **34명 전원에게 생성 가능**하였다.
- 실제 V4–V6 추적 분석에서는 해당 시점의 daily CGM 자료가 존재하는 **30명**이 포함되었다.

---

## 3-3. 결측 처리

- 걸음수와 수면시간은 기존 주분석 기준과 동일하게 **median imputation**을 적용하였다.

### 대체 순서

1. 동일 대상자-visit 중앙값
2. 동일 대상자 중앙값
3. 동일 visit 중앙값
4. 전체 중앙값

### 결측 대체 결과

| 변수 | 대체 전 결측 | 최종 결측 |
|---|---:|---:|
| 걸음수 | 417 | 0 |
| 수면시간 | 24 | 0 |

즉, lifestyle 변수는 최종 모델에서 결측 없이 사용되었다.

---

## 3-4. 모델 구성

- Baseline-adjusted GEE는 다음 5개 모델로 구성하였다.

| 모델 | 보정 변수 |
|---|---|
| B0 | cluster + visit + baseline CGM |
| B1 | B0 + age + diabetes duration + medication, sex 제외 |
| B2 | B1 + sex |
| B3 | B1 + steps + sleep, sex 제외 |
| B4 | B2 + steps + sleep |

- 이 중 **B4 모델**은  
  **baseline CGM, visit, 연령, 당뇨 유병기간, 약물 사용, 성별, 걸음수, 수면시간**을 모두 보정한 가장 완전한 모델로 해석하였다.

---

## 3-5. 결과 해석 원칙: p-value보다 effect size 중심

- 본 연구는 독립 대상자 수가 크지 않기 때문에, 결과를 해석할 때 **p-value만으로 유의성 여부를 판단하기보다 β estimate와 95% CI를 중심으로 해석**하였다.

- 특히 다음과 같은 지표는 단순한 통계적 유의성뿐 아니라 **임상적으로 어느 정도 의미 있는 크기의 차이인지**가 중요하다.

  - 평균혈당이 몇 mg/dL 높은가
  - TIR이 몇 %p 낮은가
  - TAR가 몇 %p 높은가

- 따라서 아래 결과에서는  
  1. **효과 방향**
  2. **β estimate의 크기**
  3. **95% CI의 범위**
  4. **p-value**  
  순서로 해석하였다.

# 5. Longitudinal Trajectory Analysis: 2-class LCGA 보완 분석

## 5-1. 분석 목적

앞선 3-class LCGA에서는  
V1 → V4 → V5 → V6의 CGM 변화 궤적을 분석한 결과,  
BIC 기준으로는 3-class 모형이 가장 우수하였다.

그러나 3-class 모형에서는 일부 trajectory class의 대상자 수가 **2명에 불과**하여,  
해당 class가 안정적인 잠재 궤적군이라기보다  
**소수의 극단 패턴을 분리한 결과일 가능성**이 있었다.

따라서 해석의 안정성을 보완하기 위해  
각 CGM 지표에 대해 **2-class LCGA 결과를 별도로 확인**하였다.

분석한 CGM 지표는 다음과 같다.

1. **Daily mean glucose**
2. **TIR 70–180**
3. **TAR >250**

---

## 5-2. 2-class 모형의 class 크기와 분류 안정성

### 1) Class별 대상자 수

| Outcome | Class 1 | Class 2 |
|---|---:|---:|
| Daily mean glucose | 27명 | 7명 |
| TIR 70–180 | 27명 | 7명 |
| TAR >250 | 28명 | 6명 |

3-class 모형에서는 일부 class가 2명에 불과했지만,  
2-class 모형에서는 최소 class 크기가 **6–7명**으로 증가하였다.

즉, **2-class 모형이 3-class 모형보다 class size 측면에서 훨씬 안정적**이다.

---

### 2) Posterior probability

| Outcome | Class | 평균 posterior probability | 최소 posterior probability |
|---|---:|---:|---:|
| Daily mean glucose | Class 1 | 0.973 | 0.747 |
| Daily mean glucose | Class 2 | 0.871 | 0.500 |
| TIR 70–180 | Class 1 | 0.946 | 0.649 |
| TIR 70–180 | Class 2 | 0.993 | 0.964 |
| TAR >250 | Class 1 | 0.973 | 0.791 |
| TAR >250 | Class 2 | 0.956 | 0.907 |

### 해석

- 대부분의 class에서 평균 posterior probability가 **0.87 이상**으로 높게 나타났다.
- 특히 TIR 70–180의 Class 2와 TAR >250의 Class 2는  
  평균 posterior probability가 각각 **0.993**, **0.956**으로 매우 높았다.
- 다만 Daily mean glucose의 Class 2는 최소 posterior probability가 약 **0.50** 수준으로,  
  일부 대상자는 class 경계가 다소 불명확했을 가능성이 있다.

전체적으로 보면,  
**2-class 모형의 분류 안정성은 비교적 양호한 편**으로 판단된다.

---

## 5-3. 체성분 표현형과 2-class trajectory의 연관성

각 CGM 지표에서 도출된 2-class trajectory와  
체성분 표현형 간의 연관성을 확인하였다.

체성분 표현형은 다음 3군이다.

- 근육우세·저지방형
- 복부비만·고지방형
- 저근육형

### 연관성 검정 결과

| Outcome | Fisher’s exact p-value | Cramér’s V |
|---|---:|---:|
| Daily mean glucose | 0.646 | 0.200 |
| TIR 70–180 | 0.646 | 0.200 |
| TAR >250 | 1.000 | 0.116 |

### 해석

- 세 outcome 모두에서 Fisher’s exact test는 유의하지 않았다.
- Cramér’s V도 **0.116–0.200 수준**으로 작았다.
- 따라서 현재 자료에서는  
  **체성분 표현형에 따라 longitudinal CGM trajectory class 분포가 뚜렷하게 달라진다는 근거는 확인되지 않았다.**

---

## 5-4. 체성분 표현형별 trajectory class 분포

### 1) Daily mean glucose

| 체성분 표현형 | Class 1 | Class 2 |
|---|---:|---:|
| 근육우세·저지방형 | 7 | 1 |
| 복부비만·고지방형 | 11 | 2 |
| 저근육형 | 9 | 4 |

### 해석

- 저근육형은 Class 2에 **4명**이 포함되어,  
  다른 군집에 비해 Class 2 비율이 다소 높아 보였다.
- 그러나 전체 검정에서는 유의한 연관성이 확인되지 않았다.
- 따라서 저근육형이 특정 평균혈당 trajectory에 더 많이 속한다고 단정하기는 어렵다.

---

### 2) TIR 70–180

| 체성분 표현형 | Class 1 | Class 2 |
|---|---:|---:|
| 근육우세·저지방형 | 7 | 1 |
| 복부비만·고지방형 | 11 | 2 |
| 저근육형 | 9 | 4 |

### 해석

- TIR 70–180의 class 분포는 Daily mean glucose와 동일하였다.
- 저근육형에서 Class 2 비율이 상대적으로 높게 나타났지만,  
  Fisher p-value는 **0.646**으로 통계적으로 명확하지 않았다.
- 따라서 **저근육형이 불리한 TIR 궤적에 집중적으로 속한다는 근거는 부족하다.**

### 3) TAR >250

| 체성분 표현형 | Class 1 | Class 2 |
|---|---:|---:|

| 복부비만·고지방형 | 11 | 2 |
| 저근육형 | 10 | 3 |

### 해석

- TAR >250에서도 저근육형이 Class 2에 다소 더 많이 포함되었지만,  
  전체 분포 차이는 거의 없었다.
- Fisher p-value는 **1.000**, Cramér’s V는 **0.116**으로 나타나  
  체성분 표현형과 고도 고혈당 궤적군 사이의 관련성은 매우 약했다.

## 5-5. 3-class 결과와 2-class 결과 비교

| 비교 항목 | 3-class 모형 | 2-class 모형 |
|---|---|---|
| BIC | 더 낮음 | 다소 높음 |
| 일부 class 크기 | 2명 수준의 매우 작은 class 존재 | 최소 6–7명 |
| 해석 안정성 | 낮음 | 상대적으로 높음 |
| 체성분 표현형과의 연관성 | 유의하지 않음 | 유의하지 않음 |
| 보고 전략 | 탐색적 참고 | 보완분석으로 더 안정적 |

### 해석

- **모형 적합도만 보면 3-class가 더 우수**하였다.
- 그러나 3-class 모형에서는 일부 class의 표본이 지나치게 작아  
  임상적 궤적군으로 해석하기에는 불안정성이 있었다.
- 반면 2-class 모형은 class size가 더 안정적이고 posterior probability도 전반적으로 높았다.
- 따라서 최종 보고에서는 다음과 같이 정리하는 것이 가장 적절하다.

> BIC 기준으로는 3-class 모형이 선호되었으나, 일부 class의 표본 수가 매우 작아 해석의 안정성에 한계가 있었다. 이에 따라 2-class 모형을 추가로 검토하였으며, 2-class 모형에서는 class 크기와 분류 안정성이 개선되었다. 그러나 체성분 표현형과 trajectory class 간의 연관성은 2-class와 3-class 모두에서 뚜렷하게 확인되지 않았다.

---

## 5-6. 최종 해석

이번 2-class LCGA 보완 분석의 핵심 결론은 다음과 같다.

> **2-class 모형은 3-class 모형보다 해석 안정성이 높았지만, 체성분 표현형에 따라 CGM 변화 궤적 패턴이 명확하게 달라진다는 근거는 확인되지 않았다.**

특히,

- 저근육형이 일부 불리한 trajectory class에 상대적으로 더 많이 포함되는 경향은 있었으나,
- 통계적 검정 결과와 효과크기 모두 강하지 않았고,
- 현재 표본에서는 이를 명확한 군집 간 차이로 해석하기 어렵다.

따라서 Longitudinal Trajectory Analysis는  
**체성분 표현형별 CGM 변화 패턴의 차이를 강하게 입증하는 결과라기보다,  
보완적·탐색적 분석으로 제시하는 것이 적절하다.**

## 5-9. 결론

> **Longitudinal Trajectory Analysis를 통해 CGM 지표의 종단 변화 패턴을 탐색하였으나, 체성분 표현형별로 뚜렷하게 구분되는 trajectory pattern은 확인되지 않았다.**  
> 다만 본 분석은 기존 daily-level GEE 및 baseline CGM 조정 분석과 함께, 체성분 표현형과 혈당관리 양상의 관계를 다각도로 검토했다는 점에서 보완적 의미를 가진다.
