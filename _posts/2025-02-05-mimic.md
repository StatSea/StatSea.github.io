#  MIMIC-III 기반 중환자 사망 예측 프로젝트

## 1. 프로젝트 개요

- **목표**: 중환자 ICU 데이터 기반 사망 여부 예측
- **데이터**: MIMIC-III (ICU 환자 EMR 데이터)
- **핵심 문제**
  - 시계열 데이터 처리
  - 결측치 문제
  - 환자 단위 feature 구성
- **성과**
  - AUROC: 0.84
  - AUPRC: 랜덤 대비 5배 향상

---

## 2. 데이터 이해 (MIMIC-III)

MIMIC-III는 중환자실 환자의 EMR 기반 데이터로  
환자의 생체신호, 검사 결과, 처방 등이 포함된 시계열 데이터이다.

 주요 특징:
- 환자 단위 데이터 + 시간 흐름 데이터
- 비정형 + 정형 데이터 혼합
- 다수 테이블 구조 (Relational DB)

 데이터 구조:
- PATIENTS: 환자 정보
- ADMISSIONS: 입원 정보
- CHARTEVENTS: 생체 신호
- LABEVENTS: 검사 결과

※ 해당 내용은 강의 자료 기반 :contentReference[oaicite:0]{index=0}

## 3. 데이터 전처리

### 3.1 환자 단위 데이터 구성

- SUBJECT_ID 기준으로 데이터 통합
- 시간 기준 정렬

### 3.2 시계열 처리

- 일정 시간 window 설정
- 통계값 추출:
  - mean / min / max

### 3.3 결측치 처리

- 평균 대체 / forward fill
- 변수별 결측률 분석

## 4. Feature Engineering

다음과 같은 feature를 생성:

- Vital Signs:
  - Heart Rate
  - Blood Pressure
  - Respiratory Rate
- Lab:
  - Lactate
  - pH
  - Sodium

 시계열 → 정적 feature로 변환
