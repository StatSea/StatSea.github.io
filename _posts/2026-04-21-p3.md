# 2. 체성분 상태에 따라 그 시점의 혈당 반응도 달라지는가?

## 2-1. 분석 목적
- 시험군 2의 신체계측 자료를 이용하여 체성분 패턴을 요약하고 유사한 체형/체성분 특성을 가진 대상자들을 군집으로 분류하기 위해 PCA와 군집분석을 수행하였다.

## 2-2. 사용 데이터
- 분석 표본 수: 123
- 변수 수: 21개 컬럼 기반 데이터
- 사용 visit:
  - V1: 34명
  - V4: 32명
  - V5: 30명
  - V6: 27명
- 총 123개의 방문 단위 관측치를 사용함

## 2-3. 사용한 신체계측 변수
- 허리둘레(CM)
- 체수분
- ICW
- ECW
- 단백질
- 무기질
- 체지방
- 제지방량
- 체중
- 골격근량
- BMI
- 체지방률
- 복부지방률
- 내장지방

## 2-4. 분석 방법
(1) PCA 수행
- 여러 신체계측 변수들이 서로 높은 상관을 가질 가능성이 크므로 변수들을 소수의 주성분으로 요약하기 위해 PCA를 시행하였다.
- 누적 설명력을 기준으로 주성분 개수를 선택하였다.

(2) 군집분석 수행
- PCA score를 기반으로 k-means 군집분석을 시행하였다.
- k=2~6 범위에서 silhouette score를 비교하여 최적 군집 수를 검토하였다.

## 2-5. PCA 결과
- 선택된 주성분 개수: 2개
- 누적 설명력: 0.9417 (약 94.2%)

→ 즉, 14개의 신체계측 변수 정보가 2개의 주성분만으로도 대부분 설명되었다.
→ 따라서 대상자의 체성분 특성은 비교적 단순한 축으로 요약될 수 있다고 해석할 수 있다.

## 2-6. 주성분 해석
[PC1]
- 대부분의 변수에서 양의 적재값을 보임
- 특히 체중, 체수분, ICW, ECW, 단백질, 무기질, 제지방량, 골격근량 등이 고르게 크게 적재됨
- 해석:
  → 전반적인 “몸의 크기/제지방량/근육량” 축
  → PC1이 높을수록 체격이 크고 제지방량 및 근육량이 많은 경향

[PC2]
- 체지방률(0.516), 체지방(0.385), 내장지방(0.374), 복부지방률(0.291), BMI(0.250)가 상대적으로 크게 적재됨
- 반면 체수분, ICW, ECW, 단백질, 제지방량, 골격근량은 음의 방향으로 적재됨
- 해석:
  → “지방 중심 체성분 축” 또는 “지방비율 증가 vs 제지방 중심” 축
  → PC2가 높을수록 체지방률, 내장지방, 복부지방 경향이 높은 체형으로 볼 수 있음

# 3. 체성분 그룹에 따른 CGM 지표 비교

## 3-1. 분석 목적
- 1단계에서 도출된 체성분 군집(Cluster_0, Cluster_1)에 따라
  연속혈당측정(CGM) 지표가 차이를 보이는지 확인하였다.
- 이를 통해 체성분 프로파일이 전반적인 혈당조절 상태와 관련이 있는지 평가하고자 하였다.

## 3-2. 사용 데이터
- 데이터셋: cgm_visit_summary_df
- 데이터 크기: 123행 × 17열
- 분석 단위: 등록번호 + visit 단위
- 포함 변수:
  - 등록번호
  - visit
  - cluster_name
  - mean_glucose
  - GMI
  - CV
  - TIR
  - TAR
  - TBR

## 3-3. 분석에 사용한 CGM 지표 의미
- mean_glucose: 평균혈당
- GMI: CGM 기반 추정 당화혈색소
- CV: 혈당변동계수
- TIR: 목표범위 내 혈당 비율(Time in Range)
- TAR: 목표범위 초과 혈당 비율(Time Above Range)
- TBR: 목표범위 미만 혈당 비율(Time Below Range)

## 3-4. 분석 방법
- 1단계 PCA + 군집분석에서 도출한 체성분 군집 정보를 등록번호-visit 단위 CGM 요약자료에 연결하였다.
- 각 군집별로 mean, standard deviation, median, count를 산출하여 기술통계를 비교하였다.
- 군집 간 차이 검정은 비모수 방법인 Kruskal-Wallis 검정을 이용하였다.

# 5. 탄수화물 15g 증가 당 혈당반응

- 모델 식 : outcome∼보정변수들+(1∣등록번호)+(1∣등록번호:visit)
- (1∣등록번호) : 사람마다 고유한 baseline 차이가 있음을 반영
- (1∣등록번호:visit) : 같은 사람 안에서도 visit마다 baseline이 달라질 수 있음을 반영

## 5-1. 분석 목적
- 체성분 군집에 따라 같은 양의 탄수화물을 섭취했을 때 혈당반응이 다르게 나타나는지 평가하였다.
- 특히 식사 이벤트 단위에서 탄수화물 섭취량 증가가
  iAUC_0_120, 최고혈당, 최고혈당상승량에 미치는 영향을 확인하고,
  그 효과가 체성분 군집별로 달라지는지 interaction을 검정하였다.

## 5-2. 분석 데이터 및 방법
- 분석 단위: 식사 이벤트 단위
- 군집 변수: 1단계에서 도출한 체성분 군집(Cluster_0, Cluster_1)
- 독립변수: 탄수화물 15g 단위 변수(carb_15g_c)
- 종속변수:
  1) iAUC_0_120
  2) 최고혈당
  3) 최고혈당상승량
- 보정변수:
  - 성별
  - 경구약 M1 코드
  - 나이
  - 당뇨 유병기간
  - 평균 걸음수
  - 평균 수면시간
  - 에너지(kcal)
  - 단백질(g)
  - 지질(g)
- 모형:
  - subject(등록번호) 수준 random effect
  - subject-visit 수준 nested random effect
  를 포함한 Visit 중첩 mixed model을 사용하였다. 

## 5-3. 표본 수
- iAUC_0_120 분석: n = 2240
- 최고혈당 분석: n = 2287
- 최고혈당상승량 분석: n = 2257

# 6. 탄수화물 비율 증가 당 혈당반응
-  탄수화물(g) * 4 / 에너지(kcal) * 100로 탄수화물 비율을 계산하였다.
- 식사 내 탄수화물 비율(carb_pct_recalc)이 식후 혈당반응에 미치는 영향을 평가하고,
- 그 효과가 체성분 군집(Cluster_0, Cluster_1)에 따라 달라지는지 확인하고자 하였다.
- 본 단계는 3단계의 탄수화물 절대량(g) 분석과 달리,
  “식사 구성에서 탄수화물이 차지하는 비율”에 초점을 둔 추가 분석이다.
